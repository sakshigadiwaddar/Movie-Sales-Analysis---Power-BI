# Movie-Sales-Analysis---Power-BI
# Project Overview

The Movie Sales Analysis Dashboard is an interactive Power BI project designed to analyze movie industry data and uncover insights related to movie performance, audience engagement, ratings, and revenue trends. The project transforms raw movie data into meaningful visualizations that help stakeholders understand key business metrics and make data-driven decisions.

# Objectives
Analyze movie revenue and audience engagement.
Identify top-performing genres, directors, and movies.
Track IMDb ratings and voting patterns.
Understand trends across different release years.
Create an interactive dashboard for effective data visualization.

# Tools & Technologies
Microsoft Power BI Desktop
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Data Visualization & Dashboard Design

# Dashboard Features
Total Revenue Analysis
Average IMDb Rating Analysis
Total Audience Votes Tracking
Genre-wise Revenue Distribution
Director-wise Movie Performance
Year-wise Movie Trends
Interactive Filters and Slicers
Dynamic KPI Cards

# Key Insights
Analyzed 1000+ movies from the dataset.
Generated 72.34 Billion total movie revenue.
Movies received over 170 Million audience votes.
Average IMDb rating across movies is 6.72.
Adventure and Action genres contribute the highest revenue.
Movies released between 2010–2014 received higher audience engagement and voting activity.
